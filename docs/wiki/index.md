# flowsurface 操作ガイド

flowsurface は Rust 製のデスクトップチャートアプリです。暗号資産・国内株式のリアルタイムデータを複数ペインで表示し、過去データの再生（リプレイ）と仮想売買・PnL 管理ができます。

## 目次

- [基本的な使い方](getting-started) — 起動・画面構成・ペイン操作
- [チャート](charts) — 各チャートの見方と設定
- [リプレイ](replay) — 過去データの再生・仮想取引
- [注文（立花証券）](orders) — 発注・照会・余力確認
- [設定・カスタマイズ](settings) — レイアウト保存・テーマ・タイムゾーン
