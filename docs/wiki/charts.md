# チャート

## Kline Chart（ローソク足）

### 表示できるもの

- ローソク足（OHLCV）
- インジケータ（EMA, SMA, VWAP, Bollinger Bands, RSI, MACD 等）
- ボリューム（下部）

### 設定

| 項目 | 選択肢 |
|------|--------|
| タイムフレーム | M1 / M5 / M15 / M30 / H1 / H4 / D1 / W1 / Mnth |
| Y 軸スケール | Linear / Log |
| 描画方式 | Candlestick / OHLC Bar / HLC |

### インジケータの追加

ペイン上部の設定アイコンからインジケータを追加・削除できる。各インジケータのパラメータ（期間等）はカスタマイズ可能。

## Heatmap Chart（ヒートマップ）

板情報（L2 Depth）と約定（Trades）を時系列グリッドで可視化する。

```
価格 ↑
    │    ██░░░░███░░██░░
    │    ░░░░░████████░░
    │    ░░░░░░░░░░░░░░
    │    ███░░░░░░░░░██
    └──────────────────→ 時刻
         色の濃さ = ボリューム
```

### 設定

| 項目 | 説明 |
|------|------|
| 価格グルーピング | 価格軸の丸め幅。小さいほど細かい。例: 10 = 10 円単位 |
| 時間集計 | 何秒単位でグリッドを区切るか。1 / 5 / 15 / 60 秒等 |
| ボリュームプロファイル | 「固定高さ」か「表示範囲スケーリング」か |

### リプレイ対応

Depth の履歴データは取引所 API が非提供のため、リプレイ中はヒートマップに「Replay: Depth unavailable」のオーバーレイが表示される。

## Footprint Chart（フットプリント）

ローソク足の中に、価格帯別の約定内訳を重ねて表示する。

```
┌─────────────────┐
│  3260  52 | 30  │  ← 価格帯ごとの 売約定 | 買約定
│  3250  10 | 120 │
│  3240   5 |  8  │
└─────────────────┘
```

### スタディ

| スタディ | 説明 |
|---------|------|
| Imbalance | 売買のアンバランスを強調表示（一方が他方の N 倍超のセル） |
| Naked POC | 以降のバーに再タッチされていない最多約定価格帯 |
| Clustering | 約定の密集度 |

### 対応取引所

Trades データが必要なため、現状 Binance 系が対象。

---

## Time & Sales（歩み値）

個別の約定をリアルタイムでリスト表示する。

```
時刻        価格    数量    方向
09:30:15   3,250   1,200    買
09:30:14   3,249     500    売
09:30:13   3,248     800    買
...（新着が上に追加）
```

### 機能

- 売買方向で色分け（買=青系 / 売=赤系）
- 時間を遡ってスクロール可能
- フィルター: 買のみ / 売のみ表示

---

## DOM / Ladder（板情報）

L2 板情報を価格レベルごとに表示する。

```
売気配           買気配
数量   価格   数量
 800   3,260
 500   3,258
 200   3,255
         ──── スプレッド
       3,250  1,500
       3,248    800
       3,245    300
```

右側の列に直近の約定数量も表示される。

### 価格グルーピング

価格を指定幅で丸めて板の段数を調整できる。取引所によってデフォルトのグルーピングが異なる。

### リプレイ対応

Depth の過去データは取得不可のため、リプレイ中は「Replay: Depth unavailable」が表示される。

---

## Comparison Chart（比較チャート）

複数銘柄の終値を基準時点からの変化率（%）で折れ線表示する。

```
%
+5% │    BTC ──
+3% │   ETH ─
 0% │──────────────→ 時刻
-2% │              SOL ─
```

- 各銘柄は色分けして表示
- 基準: 表示範囲の最左端の終値を 0% として正規化

> リプレイ中の複数銘柄同期は未実装。
