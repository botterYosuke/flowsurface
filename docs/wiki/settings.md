# 設定・カスタマイズ

## 設定画面を開く

左サイドバー下部の **⚙（設定）** ボタンをクリック。

---

## 基本設定

| 設定項目 | 説明 |
|---------|------|
| **タイムゾーン** | チャートの時刻表示に使うタイムゾーン。例: `Asia/Tokyo` |
| **UI スケール** | 75% / 100% / 125% / 150% / 175% / 200% |
| **テーマ** | カラーテーマの切替（Dark / Light 等）|
| **テーマカラー編集** | 各色をカスタマイズ可能 |

---

## 立花証券 の接続

1. 設定画面から **「立花証券 e支店 ログイン」** を開く
2. ユーザーID とパスワードを入力してログイン
3. セッションは keyring に保存され、次回起動時に自動復元される

> 初回利用には電話認証が必要。立花証券 e支店の仕様を確認してください。

**セッションが切れた場合**: Toast 通知が表示される。設定画面から再ログインする。

---

## レイアウト管理

### レイアウトの保存

サイドバーの **📐（レイアウト）** ボタンをクリックすると、保存済みレイアウトの一覧が表示される。

- **保存**: 現在のペイン構成・チャート設定を新しいレイアウトとして保存
- **切替**: 保存済みレイアウトをクリックして切り替える
- **削除**: レイアウトを削除する

### 自動保存

アプリ終了時・設定変更時にペイン構成・ウィンドウサイズが `saved-state.json` に自動保存される。次回起動時に自動復元される。

### saved-state.json の場所

```
C:\Users\<ユーザー名>\AppData\Roaming\flowsurface\saved-state.json
```

---

## サウンド

サイドバーの **🔊（音量）** ボタンで効果音の ON / OFF を切替できる。

約定ストリームの更新に連動してサウンドが鳴る。

---

## ポップアウトウィンドウ

ペイン右上のポップアウトボタンで、そのペインを別ウィンドウとして独立させられる。

- 別モニターに配置するのに便利
- 閉じると元のウィンドウに戻る（ペインの内容は保持される）
- ポップアウト状態もレイアウトとして保存される

---

## HTTP API（外部制御）

`127.0.0.1:9876` でローカル HTTP サーバーが起動している。E2E テストや自動化スクリプトからアプリを操作できる。

ポートは環境変数 `FLOWSURFACE_API_PORT` で変更可能。

### よく使うコマンド

```bash
# アプリ状態をファイルに保存
curl -X POST http://127.0.0.1:9876/api/app/save

# スクリーンショット（C:/tmp/screenshot.png に保存）
curl -X POST http://127.0.0.1:9876/api/app/screenshot

# ペイン一覧
curl http://127.0.0.1:9876/api/pane/list

# ペインの銘柄変更
curl -X POST http://127.0.0.1:9876/api/pane/set-ticker \
  -H "Content-Type: application/json" \
  -d '{"pane_id":"<uuid>","ticker":"BinanceLinear:BTCUSDT"}'

# ペインのタイムフレーム変更
curl -X POST http://127.0.0.1:9876/api/pane/set-timeframe \
  -H "Content-Type: application/json" \
  -d '{"pane_id":"<uuid>","timeframe":"M5"}'

# 立花証券セッション確認
curl http://127.0.0.1:9876/api/auth/tachibana/status

# Toast 通知の一覧確認
curl http://127.0.0.1:9876/api/notification/list
```

リプレイ制御の API は [リプレイ / HTTP API による制御](replay.md#http-api-によるリプレイ制御) を参照。
