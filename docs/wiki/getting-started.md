# 基本的な使い方

## 画面構成

```
┌──────────────────────────────────────────────────────────┐
│ FLOWSURFACE                       2026-04-17 14:32:05    │  タイトルバー
├──────────────────────────────────────────────────────────┤
│ [LIVE] ──────────────────────────────────────────────── │  ヘッダーバー
│ [REPLAY] 2026-04-01 09:00 ~ 15:00  ⏮ ▶ ⏭ 2x           │  (リプレイ時)
├─────────┬────────────────────────────────────────────────┤
│         │                                                │
│ サイドバー│            ペイングリッド                      │
│         │                                                │
│ 🔍 検索  │  ┌──────────────┐  ┌──────────────┐          │
│ 🖊 注文  │  │  ペイン 1    │  │  ペイン 2    │          │
│ 📐 レイアウト│  │ (KlineChart) │  │ (Heatmap)    │          │
│ 🔊 音量  │  └──────────────┘  └──────────────┘          │
│ ──────  │                                                │
│ ⚙ 設定  │                                                │
└─────────┴────────────────────────────────────────────────┘
```

## ペイン操作

### ペインの分割

1. ペイン右上の **`+`** ボタンをクリック
2. `Horizontal`（上下）または `Vertical`（左右）を選択
3. 新しいペインが **Starter（未設定）** 状態で追加される

### ペインの種類を変える

1. Starter ペインのドロップダウンから種類を選ぶ
2. 銘柄・タイムフレームを指定するとチャートが表示される

**選べるチャート:**

| 名前 | 用途 |
|------|------|
| Heatmap Chart | 板・約定をヒートマップで表示 |
| Kline Chart | ローソク足チャート |
| Footprint Chart | ローソク足 + 価格別約定内訳 |
| Time & Sales | 歩み値リスト |
| DOM / Ladder | 板情報（L2） |
| Comparison Chart | 複数銘柄の相対比較 |

> 注文パネル（Order Entry / Order List / Buying Power）はサイドバーの鉛筆ボタンから開く。Starter からは選べない。

### ペインを閉じる

ペイン右上の **`×`** ボタン。最後の 1 ペインは削除不可。

### ポップアウトウィンドウ

ペイン右上の **ポップアウトボタン** でペインを別ウィンドウとして切り離せる。元ウィンドウに戻す操作でも閉じることができる。

## 銘柄の変更

### サイドバーの検索から

1. 左サイドバーの **🔍（虫眼鏡）** をクリック
2. 検索ボックスに銘柄コードまたは名称を入力
3. 取引所フィルターで絞り込む（Binance / 立花証券 等）
4. リストの銘柄をクリック → フォーカス中のペインに反映

### ペインリンク（複数ペイン一括切替）

ペイン上部のリンクグループ（A〜I）を設定すると、同グループのペインが一括で銘柄切替される。

例: ペイン1（KlineChart）とペイン2（Heatmap）を同じグループ `A` に設定 → 検索から銘柄を変えると両方が切り替わる。

## ヘッダーバー

### LIVE モード

```
[LIVE]  2026-04-17 14:32:05
```

- 現在時刻がリアルタイムで更新される
- 再生制御ボタンはすべて無効

### REPLAY モード

```
[REPLAY]  2026-04-01 09:00 ~ 2026-04-01 15:00   ⏮ ▶ ⏭  2x   Loading...
```

- 表示時刻は仮想時刻（リプレイが進むと変わる）
- 再生制御ボタンが有効になる
- **`⏮`** : 開始位置に戻す（再生中なら停止してから移動）
- **`▶`** : 再生開始 / **`⏸`** : 一時停止 / **`▶`** : 再開（状態依存）
- **`⏭`** : 次のバーへ進む（StepForward）
- **`◀`** : 前のバーへ戻す（StepBackward）
- **`2x`** ボタン: 速度切替（1x → 2x → 5x → 10x → 1x）
- **`Loading...`** : データ取得中に表示

LIVE ↔ REPLAY の切替は [リプレイ機能](replay.md) を参照。

## キーボードショートカット

| キー | 動作 |
|------|------|
| **F5** | LIVE ↔ REPLAY 切替 |
| **Escape** | モーダル・メニューを閉じる |

---

## 起動時の自動再生

前回 REPLAY 状態で `saved-state.json` を保存した場合、次回起動時に同じ区間を自動で再生開始する。

立花証券のセッションが必要な構成でログアウトしていた場合は、Toast 通知「Replay auto-play was deferred: please log in to resume」が表示され、ログイン後に手動で Play を押す。
